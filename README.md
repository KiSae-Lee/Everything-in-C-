# CPP-Examples
CPP examples for education.

STANDARD: C++ 20

- Cmake based project.
- Open With Visual Studio.
- Make sure your Visual Studio has CMake components.

### C++ Basic knowledge
1. Overview
2. C++ Compiler
3. C++ Linker
4. Basic knowledge
    - Variables
    - Fuctions
    - Controls (Condition, Loop) | if, for, while, do-while, switch
    - Ternary Operators
    - Headers
    - Debug (Log System and Break Points)
    - Pointers
